Guía Básica de Funcionamiento

La aplicación se encarga de pedirle datos importantes a un Optical Network Terminal (ONT) específico para mostrarlos en una interfaz gráfica web que facilita su lectura y comprensión.
Para poder acceder a esos datos del ONT, es necesario pasar por el Optical Line Terminal (OLT) primero, que luego se encargará de solicitar los datos requeridos al ONT.
Los datos se guardan en una base de datos que ayuda para agilizar próximas búsquedas, ya que, si se hace una consulta con la misma ip y número de serie dentro de un rango de 24 horas, se redirigirá a la búsqueda anterior almacenada en la base de datos con el objetivo de no tener que esperar el tiempo de procesamiento que requiere una nueva consulta.

Importante: solo son soportados modelos de OLT Huawei.

Instrucciones de Uso

  1. En la página principal, encontrará dos rectángulos para poder ingresar datos.
  2. El primero, "OLT IP", se trata de la dirección ip de la OLT de la empresa.
  3. El segundo, "ONT SN", se trata del número de serie de la ONT del cliente de donde se quieren recavar datos.
  4. Ingrese los datos requeridos en los dos campos mencionados.
  5. Presione sobre el botón "Buscar".
  6. Si ya existe una consulta igual anterior en la base de datos, dentro de un rango de 24 horas, se priorizará esa búsqueda para ahorrar tiempo.
  7. Si no hay ninguna coincidencia en la base de datos, se realizará una nueva consulta, lo que puede tardar unos cuantos segundos (en ambientes de pruebas ha tardado unos 8 segundos aproximádamente).
  8. Una vez tenga la búsqueda lista, se redirigirá a la página con los resultados.
  9. La página de resultado consta de una tabla con los datos requeridos más la fecha y la hora en la que se ha realizado la consulta.
  10. Si desea actualizar los datos para que se muestren los últimos resultados, simplemente presione sobre el botón "Actualizar Datos" ubicado en la parte superior derecha de la tabla.
